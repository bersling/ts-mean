The Ultimate Google Cloud Functions Tutorial for Node.js

April 2020

At first it helps to see where Google Cloud Functions is in the larger Google Cloud ecosystem. There are the following major categories of services that the Google Cloud is offering:

  • Compute
  • Storage & Databases
  • Networking
  • Big Data
  • Cloud AI
  • Management Tools
  • Identity & Security
  • IoT
  • API Platform

Soo... pretty much everyhing that you could potentially do in a cloud. It's obvious that in this jungle of services it's not always all too easy to find what you need. In order to use any of those services, you'll first need (a) an account for the Google Cloud and (b) a project. On account creation, a project "My First Project" will be created for you. If you don't have one, just create one.
