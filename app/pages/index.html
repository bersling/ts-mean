The starter kit uses the following core technologies:

  • MongoDB or MySQL
  • Express
  • Angular2+
  • NodeJS.

That's why it's called tsmean (TypeScript + Mongo / Mysql + Express + Angular + Node.js)!

The articles about fullstack typescript can be found under "Articles". They reach from a high-level analysis of the stack to nitty-gritty tutorials about implementation. For the starter-kit, head over to the starter-kit section, or check the code directly on github at https://github.com/tsmean/tsmean. The articles section is steadily growing and the starter kit is constantly improved, so make sure to subscribe to stay up to date!
