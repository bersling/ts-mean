Frontend

On developing a frontend with Angular there are actually already tons of good resources and there's also an amazing scaffolding tool the angular-cli. What still would be useful would be a small demonstration on how to connect Angular to the backend we've built now.

We're going to scaffold the application with the angular-cli. Install it with the command 

npm install -g @angular/cli

and create a new frontend-project with 

ng new project-name
mv project-name
cd frontend
ng serve

You now have the frontend running on port 4200, go check it in your browser at http://localhost:4200.

Our small example will closely lean on the heroes tutorial from the Angular page, just strapping it to a real database and fitting it to our backend. 

ng g component create-hero
ng g component delete-hero
ng g component display-hero
ng g component display-hero-list
ng g interface resource
ng g interface hero
ng g service resource
