Introduction

This project strives to provide a state-of-the art boilerplate for the MEAN stack (MongoDB-Express-Angular-Node) fully written in typescript. Another requirement of this project is 100% coverage with unit tests (also written in typescript!) for the boilerplate code.

Basic Setup

To get started, first create the folder structure. Create your project folder 

        mkdir my-project
        cd my-project
        mkdir backend
        cd backend

Then you'll need a package.json and a gitignore:
